## Installation of This Repository

Clone the Repository,

```Shell
your@pc:~$ git clone git@github.com:Tripple-A/Calculator.git
```

Move to the downloaded folder

```Shell
your@pc:~$ cd Calculator
```

Open the index.html page in your browser.
